# 提交说明

阅读 start-kit 的 [README.md](./README.md) 和 [Prepare_Your_Submission.md](./Prepare_Your_Submission.md)，了解应（不应）修改哪些文件以及如何实现您的算法。

当实现准备好接受评测时，在本地仓库中运行以下命令添加、提交并推送所有更改：
```
$ git add [修改的/新创建的文件]
$ git commit -m "描述本次提交的消息。"
$ git push origin
```

## 评测您的算法

将实现提交到提交仓库后即可开始评测流程。 
在竞赛网站上进入「我的提交」页面。在此可查看您的参赛作品详情与提交历史（请确保已登录）。 

在「评测分支」下可选择要评测的指定分支，便于评测不同算法或变体功能。

然后点击该页面的「开始评测」按钮以评测您的新提交。

## 查看评测进度与历史提交

在「我的提交」页面可找到「我的提交」面板。
点击某条记录的详情按钮可查看详情并跟踪正在进行的评测进度。
